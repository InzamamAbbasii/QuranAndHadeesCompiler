ABBREVIATIONS

All references given without an indication of the name are to the Holy Qur'ân, the first figure representing the number of the chapter and the second the number of the verse.

Words uttered by the Holy Prophet are throughout given within inverted commas.

In the references to Hadîth collections, the first number represents the number of the book and the second the number of the chapter, except in the case of the Musnad of Ahmad ibn Hanbal where the first figure stands for the volume and the second for the page. In all references to the Mishkât, the name of the collection from which the Mishkât has taken the particular hadîth is also indicated.
